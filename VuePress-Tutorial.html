# VuePress + Github 取代 Gitbook or Blog

首先安裝

yarn add -D vuepress

寫一個簡單的 Markdown 到 docs/readme.md 裡面

Hello vuepress World

設定 package.json 的 scripts

"scripts": {
   "docs:dev": "vuepress dev docs",
   "docs:build": "vuepress build docs"
}

建立 docs/.vuepress/config.js 檔案

module.exports = {
  title: 'Flash's BLOG',
  description: 'flash guide',
  base: '/blog/',
  repo: 'https://github.com/',
  themeConfig: {
    sidebarDepth: 0,
    sidebar: [
      { 
         title: 'Home',
         children: [ '/' ]
      }
    ],
    nav: [
      { text: 'Home', link: '/' },
      { text: 'Github', link: 'https://github.com/shazi7804/terraform-manage-guide' }
    ]
  }
}

這邊解釋幾個重要項

  • title、description 沒什麼好解釋的 …
  • base：URL 的 content path，如：localhost/terraform-manage-guide/
  • repo：提供 repository 的位置給讀者參考。
  • themeConfig：VuePress 介面主要設定值。
  • sidebar：側邊欄的顯示方式，範例是使用 Multiple Sidebars。
  • sidebarDepth：預設側邊欄顯示幾個階層 (h1, h2 …)
  • nav：Navbar 會顯示在 Header 的地方。
yarn docs:dev

查看本機的 http://localhost:8080/blog/ 應該可以看到 Hello World
